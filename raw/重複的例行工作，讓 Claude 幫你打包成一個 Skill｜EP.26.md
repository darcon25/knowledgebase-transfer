你有沒有這種感覺——好不容易跑通了一個工作流，但下次又要從頭指揮一遍？  
  
這集示範：如何把你在 Claude Code 裡已經跑熟的例行工作流，讓 Claude 自己打包成一個 Skill。  
  
什麼是 Skill：  
一份封裝好的「工作手冊」。打一個斜線指令，Claude 就知道要做什麼、按什麼步驟走、用什麼工具。  
  
這集做的事：  
① 示範一個完整工作流（語音轉文字→摘要→PDF→寄信）  
② 在結尾請 Claude 把整個流程打包成一個 Skill  
③ 在全新對話中，打一個指令就重跑全程——不用再下任何說明  
  
不用自己寫 Markdown，Claude 幫你寫。下次打一個斜線，全程自動。
