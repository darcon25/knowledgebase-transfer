# LINE Bot 自動備份至 Google Drive

**類型**：Threads 貼文  
**原始位置**：`raw/threads_2026-05-04_hddvwayp.md`  
**捕獲日期**：2026-05-04  
**作者**：@chengwei0911

## 摘要

分享解決 LINE 群組圖片/影片/檔案有下載時限問題的方案：透過 LINE Bot API 自動將對話內容同步至 Google Drive，並在 Google Sheets 建立索引，不需租用伺服器。

## 關鍵要點

- 核心工具：LINE Developers 帳號 + LINE Bot API
- 不需租伺服器，通常透過 Google Apps Script（GAS）作為中繼
- 替代儲存選擇：Telegram（利用其免費空間）或公司 NAS（私有雲）
- 進階應用：串接訂單系統與電子發票，將 LINE 升級為工作流入口
- 潛在風險：自動化行為流量過大可能違反 LINE 政策，需注意帳號封鎖風險
