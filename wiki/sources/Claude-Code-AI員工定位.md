# Claude Code 作為 AI 員工的定位

**類型**：Threads 貼文  
**原始位置**：`raw/threads_2026-05-04_2oh8tf9q.md`  
**捕獲日期**：2026-05-04

## 摘要

介紹 Claude Code 的定位與差異：不是聊天機器人，而是能直接讀取、修改檔案並執行指令的開發環境代理層。透過 CLAUDE.md 與 Skills，可建立具備專案記憶、可控且可重複協作的「AI 員工」。

## 關鍵要點

- Claude Code 支援 1M token 超長上下文，能處理完整開發週期（Debug、重構、測試、Code Review）
- **CLAUDE.md**：專案規則說明書，讓 AI 快速理解開發規範
- **生態系三分法**：Claude AI（雲端對話/想）、Claude Code（終端機代理/寫）、Claude Cowork（桌面代理/操作電腦）
- 建議設定權限邊界：刪檔、部署等高風險操作需人工確認
- 根據任務難度切換模型：日常用 Sonnet（節省成本），困難重構用 Opus
