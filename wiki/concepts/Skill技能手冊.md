# Skill（技能手冊）

Skill 是 Claude Code 中一種封裝好的工作指令文件（`skill.md`），內容是寫給 AI 讀的步驟化工作手冊。只要呼叫一個斜線指令，Claude 就會按照手冊裡定義的流程自動完成任務，不需要使用者重複下達說明。

## 重點

- **本質**：一份 Markdown 格式的提示詞文件，定義角色、任務與逐步流程
- **主檔 + 副檔**：主檔（skill.md）可建立索引，連結到更詳細的副檔（如知識庫、腳本），模型按需載入
- **觸發方式**：
  - 手動：在對話框打 `/skill名稱`
  - 自動：Claude 讀到描述後判斷情境需要時自動啟動
- **製作方式**：先手動帶 Claude 完整跑一遍工作流，跑通後命令它「把這個 session 的工作流包成 Skill」，無需自己寫程式碼
- **可複用性**：一次封裝，未來每次只需打一個指令即可重跑相同流程

## 應用場景

- 重複性例行工作（如每日回覆論壇、整理會議紀錄）
- 需要固定品質標準的任務（如去 AI 味寫作）
- 多步驟工作流自動化
