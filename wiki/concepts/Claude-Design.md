# Claude Design

Claude Design 是 Anthropic 推出的 AI 設計整合功能，讓使用者可以上傳品牌資產後直接生成符合品牌風格的設計稿，並支援 PPTX 格式匯出。

> **白話說明**：把你的 Logo、字型和配色上傳給 Claude，它就能做出符合你品牌風格的簡報或設計，最後直接存成 PowerPoint 格式讓你繼續編輯。

## 重點

- 支援上傳品牌資產（Logo、字型、配色方案）或連結 GitHub codebase，讓 AI 生成風格一致的設計
- **Edit 模式**：直覺調整個別元素
- **Comment 模式**：批次修改多個元素
- 可直接匯出可編輯的 PPTX 格式
- 支援 Prototype（原型製作）與 Design System 整合

## 趨勢意涵

- Claude 正從「文字工具」擴展到設計工作的上游，整合 AI、設計、開發的全流程
- 對已訂閱 Claude Pro 的用戶可替代 Gamma 等專門 AI 簡報工具，減少重複訂閱
- 趨勢：AI 大廠整合型功能正逐步取代單功能套殼工具
