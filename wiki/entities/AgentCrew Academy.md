# AgentCrew Academy

AgentCrew Academy 是一個專為非技術人員設計的 Claude Code 實戰教學頻道／課程平台，由講師 Dustin 主持。課程著重於實際工作流示範，不需要寫程式碼。

## 重點

- 定位：非技術人員的 Claude Code 實戰課
- 主講：Dustin
- 課程報名：https://agentcrew.cc
- 教學風格：示範實際工作場景，強調「做到跑通」再討論技術原理

## 已知課程主題

- EP.26：將例行工作流打包成 Skill
- EP.24：把書的知識變成 AI 顧問
- EP.23：會議錄音一次出三份文件
